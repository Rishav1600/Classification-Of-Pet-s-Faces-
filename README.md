# Classification-Of-Pet-s-Faces-
Developed a deep learning model for pet breed classification using a CNN in TensorFlow/Keras. The project involved extensive image preprocessing via NumPy and Pandas to prepare facial image datasets. The trained model successfully classifies various breeds (e.g., American Pitbull Terrier), in computer vision and machine learning model Development.


Project Impact:-
1)High-Accuracy Classification:- Developed a Deep Learning model to classify complex pet breeds (e.g., American Pitbull Terrier, American Bulldog, and Egyptian Mau).

2)Optimized Performance:- Leveraged Transfer Learning with pre-trained CNN architectures to enhance feature extraction, significantly reducing training time while maintaining high precision.

3)Robust Preprocessing:- Utilized Scikit-learn and NumPy for advanced data augmentation and normalization, ensuring the model is resilient to variations in image lighting and pet poses.

Technical Implementation:-
1)This project utilizes a sophisticated Computer Vision pipeline to move from raw pixel data to breed identification.

20Model Architecture:- Implemented using TensorFlow and Keras, featuring custom dense layers added atop a pre-trained feature extractor.

3)Validation Strategy:- Employed Scikit-learn for rigorous model validation and feature extraction to ensure robust identification across diverse datasets.

4)Data Pipeline: Engineered a glob-based image loading system to handle large batches of high-resolution .jpg images efficiently.

Tech Stack
1)Frameworks:- TensorFlow, Keras, Scikit-learn

2)Data Handling:- NumPy, Pandas, Glob

3)Visualization:- Matplotlib (Image & Loss curve plotting)

4)Cloud/Deployment:- Prepared for AWS (utilizing skills from AWS Academy Graduate certification).

Results & Metrics:-

1)Categorical Accuracy:- Measuring the percentage of correctly identified breeds.

2)Loss Optimization:- Monitoring training vs. validation loss to ensure the model does not overfit.

3)Visual Verification:- Implemented a display pipeline to visualize predicted labels against actual pet images.
